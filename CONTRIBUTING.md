# योगदान / Contributing

श्रुतधारा एक खुला स्वाध्याय-स्मारक है — योगदान के तीन सहज मार्ग:

1. **अशुद्धि-सूचना** — कोई भूल दिखे तो [अशुद्धि-सूचना issue](../../issues/new?template=ashuddhi-report.yml)
   खोलें, या सीधे सम्बन्धित फ़ाइल सुधार कर PR भेजें। एक पंक्ति के सुधार से
   पृष्ठ, PDF और साझा-कार्ड सब स्वतः पुनर्निर्मित होते हैं।
2. **टंकण-सेवा (नया मूल पाठ)** — किसी ग्रन्थ का मूल पाठ public-domain संस्करण से
   टाइप करें; प्रारूप [shastra/README.md](shastra/README.md) में। पहले
   [नया-पाठ issue](../../issues/new?template=naya-paath.yml) खोल लें ताकि दोहराव न हो।
3. **अर्थ/अनुवाद-परतें** — केवल अधिकार-मुक्त स्रोतों से; `@अर्थ` प्रारूप वही README बताता है।

## नियम-सार

- मूल प्राचीन पाठ (प्राकृत/संस्कृत/अपभ्रंश/पुरानी हिन्दी) public domain है —
  पर **किसी संस्करण की भूमिका, अर्थ, टीका या अनुवाद** पर सम्पादक/प्रकाशक का अधिकार
  हो सकता है। संदेह हो तो मूल-मात्र लें और स्रोत `source:` में लिखें।
- वर्तनी मूल के अनुसार रखें — "सुधार" न करें; अस्पष्ट स्थान ⚠ से चिह्नित करें।
- स्थानीय जाँच: `npm install && npm run build && npm run serve`

“इनका अध्ययन और स्वाध्याय ही आत्मकल्याण का मार्ग है।”
